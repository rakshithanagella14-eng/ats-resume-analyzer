# ATS Resume Analyzer using Python and Flask
Live demo:
https://ats-resume-analyzer-gtnq.onrender.com

A Python-based web application that analyzes resumes against job descriptions and calculates an ATS compatibility score.

## Features and Process
- Extracts text from PDF resumes
- Detects technical skills
- Compares resume skills with job description
- Calculates ATS compatibility score
- Displays matched and missing skills

## Tech Stack Used
- Python
- Flask
- PyPDF2

## How to Run This

1. Clone repository
2. Install dependencies

pip install -r requirements.txt

3. Run application

python main.py 

(if the above path not working try the following)
main.py/sample.py

(still not worked try the below)
python sample.py

4. Open browser

http://127.0.0.1:5000
